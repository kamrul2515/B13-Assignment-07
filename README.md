# 🤝 Friend Tracker App

## 📌 Project Overview

Friend Tracker App is a simple and responsive web application that helps users maintain meaningful relationships by tracking interactions with friends. You can view friend profiles, log interactions (call, text, video), and monitor your communication goals easily.

## ✨ Key Features

### 🔝 Navbar

* Fully responsive navigation bar
* Includes Home, Timeline, and Stats pages
* Active route highlighting with icons

### 🎯 Home Page

* Clean banner with title, subtitle, and action button
* Summary cards for quick insights
* Friend list displayed in a responsive grid

### 👫 Friends Section

* Dynamic friend cards loaded from JSON data
* Displays profile image, name, tags, status, and last contact info
* Clickable cards to view detailed profile

### 👤 Friend Details Page

* Two-column layout
* Left: Profile info, bio, email, status, and action buttons
* Right: Stats, relationship goal, and quick check-in options

### ⚡ Interaction Logging

* Log Call, Text, and Video interactions
* Automatically updates Timeline
* Toast notifications on actions

### 📜 Timeline Page

* Displays all interaction history
* Includes date, type, and title
* Filter by Call, Text, or Video

### 📊 Stats Page (Analytics)

* Visual pie chart showing interaction distribution
* Built using Recharts

## 📂 Data Structure

Friend data is stored in a local JSON file with realistic user information including:

* Name, email, and profile picture
* Contact frequency and status
* Tags and personal bio
* Next due date and goal
